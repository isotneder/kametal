# Atatürk İlkeleri Web Sitesi

GitHub Pages üzerinde doğrudan yayınlanabilecek, tek sayfalık ve animasyonlu bir tanıtım sitesi.

## İçerik

- `index.html`: Sayfanın ana yapısı
- `styles.css`: Tasarım, animasyonlar ve responsive düzen
- `script.js`: Scroll animasyonları, sayaçlar ve etkileşimli ilke paneli
- `favicon.svg`: Tarayıcı sekmesi ikonu

## Yayınlama

1. Bu dosyaları GitHub reposunun kök dizinine yükle.
2. GitHub'da repo ayarlarından `Pages` bölümüne gir.
3. `Deploy from a branch` seç.
4. Branch olarak `main` ve klasör olarak `/ (root)` seç.
5. Kaydettikten sonra birkaç dakika içinde siten yayınlanır.

## Yerelde Açma

`index.html` dosyasını tarayıcıda açman yeterli.
